Changes

Version 5.1.0 (2025-07-12)

  • Correct some errors in the "Exalted art Thou" prayer in the Fast category
  • Some fixes to Spanish translations

Version 5.0.0 (2023-12-10)

  • Added all the newly translated English prayers from the past 3 years from bahai.org. All the new prayers and prayers that were previously under the 'Recently Added' category have been moved to specific categories.
  • Added the Spanish version of the prayer for America
  • Dark mode support!

Version 4.1.0 (2020-09-06)

  • Many more and updated Spanish translations from the Panel Internacional de Traducción de Literatura Bahá'í al Español
  • Minor correction to a Persian prayer

Version 4.0.0 (2018-02-03)

  • Support for iPhone X* devices (notched screens)
  • Increased minimum iOS version to 10.3
  • Fixed a bug in the recents list where it would often get accidentally erased.
  • Corrections to several English prayers
  • Added a new category named Recently Added that contains newly translated English prayers
    • 12 prayers from Bahá'u'lláh
    • 19 prayers from 'Abdu'l-Bahá
  • Updates and additions to localizations of the app's interface in many languages
